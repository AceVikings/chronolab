# ChronoLab website

The public ChronoLab landing page uses Vite, React, TypeScript, Tailwind CSS, and locally bundled IBM Plex fonts. It has no authentication, backend, analytics, or runtime font requests.

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

The production output is written to `dist/`. Product copy should stay aligned with the CLI behavior documented in [`../package/README.md`](../package/README.md).
