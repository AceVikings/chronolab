# Code of conduct

We are committed to a welcoming, respectful, and harassment-free community.

Be constructive, assume good intent, respect different levels of experience, and focus criticism on ideas and code rather than people. Harassment, discrimination, threats, and disclosure of others' private information are not acceptable.

Project maintainers may edit or remove contributions and restrict participation when behavior threatens a safe and productive community. Report conduct concerns privately through the repository maintainers.
