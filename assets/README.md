# ChronoLab logo

The mark combines a wall clock with a forward arrow: the clock represents controlled realtime, while the arrow represents deterministic advancement. Its geometry remains legible at favicon size and uses only ChronoLab's documented warm-charcoal, ink, and clock-amber tokens.

- `logo-mark.svg` — primary square mark for avatars, favicons, and compact navigation
- `logo-wordmark.svg` — horizontal mark for repository and marketing headers
- `logo-mask.svg` — monochrome Safari pinned-tab mask

Keep clear space equal to one quarter of the mark width. Do not recolor the arrow separately from the amber clock ring, add gradients, rotate the mark, or place it directly on a low-contrast warm-charcoal surface without its containing square.
