# ChronoLab brand

Status: active

## Direction

Retro-futurist developer infrastructure with an industrial terminal core. Spacious marketing rhythm surrounds compact, precise command surfaces. The palette should feel like a time machine control panel: cosmic, energetic, and unmistakable without becoming toy-like.

## Mark

The primary mark is a wall-clock ring whose hand resolves into a forward arrow. It represents application realtime under control and deliberate forward advancement. Source assets and usage rules live in [`assets/`](./assets).

## Palette

- Canvas: deep-space indigo (`oklch(0.145 0.05 275)`)
- Surface: instrument blue (`oklch(0.225 0.075 275)`)
- Raised surface: illuminated indigo (`oklch(0.29 0.09 275)`)
- Ink: bright warm cream (`oklch(0.985 0.02 95)`)
- Muted ink: high-contrast parchment (`oklch(0.84 0.035 90)`)
- Accent: signal tangerine (`oklch(0.75 0.235 40)`)
- Success: verification mint (`oklch(0.84 0.16 155)`)

Tangerine represents logical time and primary action. Mint is reserved for verified success. Indigo supplies depth rather than behaving as a second accent; no additional decorative hues compete with the clock signal.

## Typography

- Interface and display: IBM Plex Sans Variable
- Commands, timestamps, and technical labels: IBM Plex Mono
- Display headings use tight tracking and restrained weight.

## Voice

Direct, technical, calm, and specific. Prefer verifiable statements such as “Host clock untouched” over aspirational marketing language.

## Motion

Crisp and functional. Use 100–150ms property-specific transitions with a custom ease-out curve. Disable non-essential motion when reduced motion is requested.
